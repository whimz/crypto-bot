Торговая стратегия Crypto Bot — контекст и обсуждения
Общая концепция
Автоматический торговый бот для Binance без фиксированных ценовых диапазонов. В отличие от стандартных grid-ботов Binance которые останавливаются когда цена выходит за установленные границы — наш бот работает на относительных индикаторах и торгует при любой цене.
Торгуемые активы: BTC, ETH, LTC (спот)

Режим: Binance Testnet (виртуальные деньги) на старте

Выбор стратегии
Почему RSI + EMA + MACD
Рассматривались варианты:

Grid-бот с фиксированными диапазонами → отклонён (останавливается за границами)
AI (Claude) как генератор торговых сигналов → отклонён
Чистый технический анализ → выбран

Почему AI не используется в торговой логике:

Latency 2-5 сек на запрос — индикаторы считаются за миллисекунды
Стоимость — каждый запрос к Claude API стоит денег, RSI/EMA бесплатны
Предсказательная сила — Claude не видит стакан, не знает инсайды, не имеет данных реального времени. На спокойном рынке его решения совпадают с тем что скажут индикаторы — просто дороже и медленнее

Где Claude реально полезен: как фильтр аномалий и для адаптивного выбора стратегии (см. ниже) — не для генерации сигналов.

Торговая логика
Индикаторы
RSI (14) — индекс относительной силы:
< 35 → актив перепродан → сигнал к покупке
> 65 → актив перекуплен → сигнал к продаже
35-65 → нейтрально → держим
EMA(50) — экспоненциальная скользящая средняя:
Цена выше EMA50 → восходящий тренд → только покупки
Цена ниже EMA50 → нисходящий тренд → только продажи
MACD (12, 26, 9) — схождение/расхождение скользящих:
Пересечение снизу вверх → растущий импульс
Пересечение сверху вниз → падающий импульс
Используется для подтверждения сигнала и расчёта confidence
Два таймфрейма — ключевое решение
Сигнал считается сильным только когда оба таймфрейма согласны:
BUY:  RSI < 35 на 15м И на 1ч + цена выше EMA50
SELL: RSI > 65 на 15м И на 1ч + цена ниже EMA50
HOLD: любое несоответствие между таймфреймами
Это фильтрует ложные сигналы на мелких колебаниях.
Confidence score
База: 60
+до 30 за глубину перепроданности/перекупленности RSI
+10 если MACD подтверждает направление на обоих ТФ
Максимум: 100

Исполняем ордер только если confidence > 70

Риск-менеджмент
Трейлинг стоп-лосс 7%
Обычный стоп: купил по $100, стоп на $93 — не меняется
Трейлинг: купил по $100, цена выросла до $150
→ стоп автоматически $139.50 (-7% от пика)
→ фиксируем больше прибыли
Урок из анализа конкурентов (3Commas, Pionex) — трейлинг стоп значительно лучше фиксированного.
DCA (Dollar Cost Averaging) — усреднение
Не покупаем всё сразу — покупаем частями
Максимум 3 усреднения подряд без продажи
Это снижает среднюю цену входа при падении
Динамический размер ордера
Волатильность низкая  (RSI 35-50) → 15% депозита
Волатильность средняя (RSI 20-35) → 10% депозита
Волатильность высокая (RSI < 20)  → 5% депозита
Лимиты аллокации
Максимум 40% депозита на одну монету
Глобальный стоп: депозит просел на 20% → пауза покупок
Panic button
/stop в Telegram → бот останавливается
/closeall → закрыть все позиции немедленно

Обсуждённые темы и решения
Скальпинг / мелкие колебания
Вывод: нецелесообразно на текущем депозите
Комиссия Binance 0.1% × 2 = 0.2% за сделку
10 сделок/день = $2 только комиссий на $100 депозите
Нужна точность > 60% что сложно на 5м RSI
Имеет смысл от $1000+ депозита
WebSocket vs REST API
Вывод: REST достаточно для текущей стратегии
REST (сейчас): опрос каждые 15 минут
WebSocket: данные в реальном времени

Для RSI на 15м свечах разница минимальная —
свеча закрывается раз в 15 минут в любом случае

WebSocket имеет смысл для:
1. Стоп-лосс в реальном времени (реальное преимущество)
2. Торговля на 1м-5м таймфреймах
Гибридный подход (запланирован):

Торговые сигналы через REST, мониторинг стоп-лосса через WebSocket ticker stream.
Фьючерсы
Вывод: интересно, но преждевременно
Медвежий рынок + SHORT стратегия = логично
Плечо ×2 — относительно безопасное

Риски:
- Ликвидация при движении 50% против позиции
- Фандинг рейт каждые 8 часов:
  0.01-0.1% от позиции → может съесть прибыль
  При удержании позиций неделями — критично

Решение: учитывать фандинг рейт при принятии решений
Когда делать: после проверки спот стратегии
Адаптивная стратегия с Claude
Вывод: самая интересная идея, запланирована
Сейчас: фиксированные пороги RSI 35/65
Адаптивно: Claude анализирует рыночный контекст
и выбирает оптимальный пресет стратегии

Входные данные для Claude:
- Волатильность за 7 дней
- Общий тренд (выше/ниже EMA200)
- Fear & Greed Index
- Объём торгов

Пресеты:
Conservative: RSI 30/70, confidence 75 — падающий рынок
Moderate:     RSI 35/65, confidence 70 — боковик (текущий)
Aggressive:   RSI 40/60, confidence 60 — растущий рынок

Cooldown: менять стратегию не чаще раза в день
Это место где Claude реально добавляет ценность
